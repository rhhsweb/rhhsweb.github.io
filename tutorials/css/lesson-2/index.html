Simple Selectors

Let's go over the simple selectors of CSS. All other selectors build off of simple selectors. As a quick reminder, CSS selectors are used to select which HTML elements you want to style. 

selector {
    property: value;
}

Element Selectors

Element selectors are used to select all of a certain type of HTML elements. An element selector simply states the element name. 

element {
    /* CSS styles go in here. */
}

For example, if you want to select all <h2> elements and bold them, it would look like this: 

/* CSS */
h2 { /* select all h2 elements. */
    font-weight: bold; /* bold the selected element(s). */
}

ID Selectors

ID selectors select the single HTML element with an ID. It cannot select more than one element since IDs must be unique for each element. An ID selector starts with a # followed by the ID name. 

#id-name {
    /* CSS styles go here. */
}

For example, if you want to only bold the second <p> element, it would look like this: 

/* CSS */
#code-block-2 #paragraph-2 {
    font-weight: bold;
}
<!-- HTML -->
<h1 id="page-title">My TED Talk</h1>
<p id="paragraph-1">Cats are great.</p>

<h2 id="subheading-1">The Truth</h2>
<p id="paragraph-2">Dogs are better.</p>

<h2 id="subheading-2">Conclusion</h2>
<p id="paragraph-3">Thank you for coming to my ted talk.</p>

In practice, you do not want to give all your elements IDs. IDs are usually only given to elements that need them.

Class Selectors

Class selectors select all HTML elements that have a particular class. A class selector starts with a . followed by the class name. Since elements can have multiple classes, they can be affected by multiple class selectors. 

/* CSS */
.red { /* select elements with the 'red' class. */
    color: red; /* give selected elements red text. */
}

.bold { /* select elements with the 'bold' class. */
    font-weight: bold; /* give selected elements bold text. */
}
<!-- HTML -->
<h1 class="red bold">Welcome to my Website!</h1>
<p class="red">Hello website visitors! How do you do?</p>

<h2 class="bold">About Me</h2>
<p class="red">I am a happy web developer!</p>

Pseudo-class Selectors

Pseudo-class selectors select elements that are in a certain state. For example, when an element is being hovered. A pseudo-class selector actually extends the other simple selectors. 

element:hover {

}

.class-name:hover {

}

#id-name:hover {
    
}

The pseudo-class selector overrides any default styling. When the selected element is no longer in the state, it reverts back to the default styling. 

#my-paragraph {
    color: blue;
}

#my-paragraph:hover {
    color: red;
}
<p id="my-paragraph">Hover me!</p>

Hover me!

Some elements may have more states than other elements do. For example, the anchor element.

Styling Anchor Elements

Anchor elements have four relevant pseudo-class selectors: :link, :visited, :hover, and :active. 


a:link {
    color: red;
}

a:visited {
    color: green;
}

a:hover {
    color: orange;
}

a:active {
    color: yellow;
}

Style Priority

CSS files are run from top to bottom. If the same selector is used multiple times throughout the file, the bottom one takes priority. 

#id-name-1 {
    color: blue; /* Ignored */
}

#id-name-1 {
    color: green; /* Ignored */
}
    
#id-name-1 {
    color: red; /* This takes priority */
}

If multiple element, class, and ID selectors target an element, the class will take priority over element selectors and ID will take priority over class selectors. 

#paragraph-1 {
    color: red; /* This takes priority */
}

p {
    color: blue; /* Ignored */
}

.paragraph {
    color: green; /* Ignored */
}
<p class="paragraph" id="paragraph-1">Some text</p>
