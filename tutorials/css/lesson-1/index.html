Adding Styles

There are three ways to add or link styles to an HTML webpage.

Inline Styling

The first way to add CSS is with inline styling. This involves putting styling inside of an element's style attribute. The following example will change the text color of a <p> element. 

<p>The quick brown fox jumped over the lazy dog.</p>
<p style="color: red;">The quick brown fox jumped over the lazy dog.</p>
<p style="color: blue;">The quick brown fox jumped over the lazy dog.</p>

The quick brown fox jumped over the lazy dog.

The quick brown fox jumped over the lazy dog.

The quick brown fox jumped over the lazy dog.

This is not a recommended way to use CSS, since it makes it harder to work with and manage. When CSS and HTML get mixed together, it becomes harder to make changes to either. Additionally, inline styling does not allow you to use all the features of CSS. To apply the same style to all <p> elements, you would have to give each one red text, you would have to change the inline styling for each element. 

<p style="color: red;">This is the first paragraph.</p>
<p style="color: red;">And this is the second paragraph.</p>
<p style="color: red;">This is the final one.</p>

This is the first paragraph.

And this is the second paragraph.

This is the final one.

Internal Stylesheets

Another way to add CSS to a webpage is with internal stylesheets. Internal stylesheets can be added using the <style> element. Styling should be placed within the head of the webpage. 

<!DOCTYPE html>
<html>
    <head>
        <style>
            p {
                color: red;
            }
        </style>
    </head>
    <body>
        <p>This is the first paragraph.</p>
        <p>And this is the second paragraph.</p>
        <p>This is the final one.</p>
    </body>
</html>

This is the first paragraph.

And this is the second paragraph.

This is the final one.

Internal stylesheets allow the full use of CSS, unlike inline styling. While it does keep HTML and CSS code relatively separate, it does not let you reuse the same styles on a different HTML page.

External Stylesheets

The final way to add CSS is with external stylesheets. Completely separate CSS documents can be linked to an HTML page using the <link/> element within the document head. 

<!-- index.html -->
<!DOCTYPE html>
<html>
    <head>
        <link rel="stylesheet" href="styles.css">
    </head>
    <body>
        <p>This is the first paragraph.</p>
        <p>And this is the second paragraph.</p>
        <p>This is the final one.</p>
    </body>
</html>
/* styles.css */
p {
    color: red;
}

This is the first paragraph.

And this is the second paragraph.

This is the final one.

External stylesheets allows you to keep CSS code completely separate from HTML. And, it allows you to reuse styles across multiple webpages just by linking it. External stylesheets is the preferred way to add CSS to a webpage.
