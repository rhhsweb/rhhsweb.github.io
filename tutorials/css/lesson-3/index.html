Color

There are many things to color and multiple ways to manipulate colors with CSS.

Background

The background-color property is used to change the background color of an element. 

h2 {
    background-color: yellow;
}
<h2>Hi</h2>

Background Image

The background-image property is used to set the background to an image. 

div {
    background-image: url(https://i.pinimg.com/originals/ce/cc/88/cecc88bacecba631d1bc7a5272313f1d.jpg)
}
<div>
    <h2>Hi</h2>
    <p>Hello there</p>
</div>

There are also many properties relating to whether the background should repeat, stretch, and more.

Gradient

The linear-gradient is a function that can be used as the value of the background-image property to create a gradient background. 

background-image: linear-gradient(angle, color1, color2, ...);
.gradient-1 {
    background-image: linear-gradient(90deg, red, yellow);
}

.gradient-2 {
    background-image: linear-gradient(180deg, cyan, yellow);
}

.gradient-3 {
    background-image: linear-gradient(20deg, red, yellow, green);
}

.rainbow {
    /* Use as many colors as you want */
    background-image: linear-gradient(90deg, red, orange, yellow, green, blue, violet);
}

You can also set at which point the colors should be placed. The percentage defines at what point in the gradient that the color is complete and not mixed. 

.gradient-4 {
    background-image: linear-gradient(
        90deg, 
        red 0%,
        yellow 10%,
        green 100%
    );
}

These are the positions where the colors are unmixed:

You can also create a repeating linear radient. Each repetition ends at the point that the last color is unmixed. 

.repeating-gradient {
    background-image: repeating-linear-gradient(
        90deg,
        red 0%,
        yellow 10%,
        green 15%
    );
}

Types of Color Values

So far, we have only been using the color keywords names. But, there are other ways to define colors. 

#id-1 {
    background-color: lightskyblue; /* color keywords */
}

#id-2 {
    background-color: rgb(255, 0, 100); /* RGB() values */
}

#id-3 {
    background-color: #a9d22e; /* 6-digit HEX codes */
}

#id-4 {
    background-color: #a7c; /* 3-digit HEX codes, (#aa77cc) */
}

#id-5 {
    /* hsl() values (hue, saturation, lightness)
        Hue is 0 to 360.
        Saturation is 0% to 100%.
        Lightness is 0% to 100%.
    */
    background-color: hsl(39, 100%, 50%);
}

You can also use rgba() if you want to make the color transparent. The a or alpha value can be a value from 0 (transparent) to 1 (opaque). 

.background {
    background-color: cyan;
}

.class-1 {
    background-color: rgba(255, 0, 0, 0.2);
}

.class-2 {
    background-color: rgba(255, 0, 0, 0.4);
}

.class-3 {
    background-color: rgba(255, 0, 0, 0.6);
}

.class-4 {
    background-color: rgba(255, 0, 0, 0.8);
}
<div class="background">
    <div class="class-1"></div>
    <div class="class-2"></div>
    <div class="class-3"></div>
    <div class="class-4"></div>
</div>

Filter

The filter property can be used to change all the colors within an element, including images. See MDN Web Docs: filter for more filters. 

.filtered {
    filter: blur(5px);
    filter: brightness(1.1); 
    filter: hue-rotate(100deg);
}

Opacity

The opacity property changes the opacity of an element and all its contents. Opacity should be a value from 0 to 1. 

.wrapper {
    background-color: yellow;
    opacity: 0.4;
}
<div class="wrapper">
    <h1>Hello there</h1>
    <p>The quick brown fox jumps over the lazy dog.</p>
</div>
