Forms

The <form> element allows you to send information to a server. 

<form method="get" action="">
    <label for="first-name">Enter your first name: </label>
    <input type="text" id="first-name" name="first-name" placeholder="John" required>
    <br>
    <label for="last-name">Enter your last name: </label>
    <input type="text" id="last-name" name="last-name" placeholder="Smith" required>
    <br>
    <label for="email-address">Enter your email: </label>
    <input type="email" id="email-address" name="email-address" placeholder="johnsmith@gmail.com">
    <br>
    <input type="reset" value="Reset Form">
    <input type="submit" value="Submit Form">
</form>



Form Attributes

There are two important form attributes that relate to sending form information: action and method.

The action attribute controls which url the information is sent to. 

<form action="index.html">
    <!-- form content -->
</form>

The method attribute specifies the HTML method to send the request with. The two main values are get and post. The get method encodes the form data into the action URL. So, get is generally used for less sensitive information. 

<form method="get" action="index.html">
    <!-- form content -->
</form>

Additionally, all inputs (except the submit button) should have a name attribute to identify it when the form information gets submitted. 

<input type="number" id="user-age">

Input Attributes

There are three important attributes for all input elements in a form: type, id, and name. The type attribute is used to specify the input type (as seen in the example above). The id attribute is used in combination with the <label> element. The <label> element is meant to describe what an input is for. 

<label for="fruit-input">What is your favourite fruit:</label>
<input type="text" id="fruit-input" name="fav-fruit">

The name attribute is used for naming the input value when it gets sent to the server. If the input does not have a name attribute, then the input value will not be sent to the server.

There is also the boolean required attribute which can be added to prevent the user from submitting the form without filling out an input field. 

<form>
    <input type="text" required>
    <input type="submit" value="Submit">
</form>

Text Input

The text input is an input for getting short text. 

<input type="text" id="username">

The placeholder attribute let's you set suggestions or examples for an input. 

<label for="order-details">Enter your order type and color:</label>
<input type="text" id="order-details" name="order-details" placeholder="double purple">

The minimum and maximum length of text can be set with the minlength and maxlength attributes. 

<form>
    <input type="text" minlength="5" maxlength="8">
    <input type="submit" value="Submit">
</form>

Radio Input

The radio input is used for selecting one of many options, like a multiple choice question. Radio inputs are linked together by having the same name attribute. 

<div>
    <p>Select your drink size.</p>
    <input type="radio" name="drink-size" id="size-1" value="small">
    <label for="size1">Small</label>

    <input type="radio" name="drink-size" id="size-2" value="medium" checked>
    <label for="size2">Medium</label>
    
    <input type="radio" name="drink-size" id="size-3" value="large">
    <label for="size3">Large</label>
</div>

Select your drink size.

The boolean checked attribute can be added to pre-select one radio input.

Checkbox Input

The checkbox input is used for selecting multiple options. 

<div>
    <p>What music course are you taking this semester?</p>
    <input type="checkbox" name="course" id="course-1" value="AMI"> 
    <label for="course1">instrumental music (AMI)</label><br>

    <input type="checkbox" name="course" id="course-2" value="AMR"> 
    <label for="course2">repertoire (AMR)</label><br>

    <input type="checkbox" name="course" id="course-3" value="AMV"> 
    <label for="course3">vocal music (AMV)</label><br>
</div>

Which music course(s) are you taking this semester?




Dropdown Select Element

The <select> element is used for selecting an option from a menu of options. Each <option> is nested within the <select> element. 

<label for="drink-select">Choose a drink:</label>
<select name="drink" id="drink-select">
    <option value="" disabled selected> -- Select an option -- </option>
    <option value="hot-coffee">Hot Coffee</option>
    <option value="iced-coffee">Iced Coffee</option>
    <option value="latte">Latte</option>
    <option value="hot-chocolate">Hot Chocolate</option>
</select>

The disabled and selected boolean attributes can be added to one of the <option> elements to be used as a message.

Textarea

The <textarea> element is used to collect large pieces of text. The size of text area can be set using the cols and rows attributes. A placeholder attribute can also be added. 

<textarea rows="5" cols="40" name="bio" placeholder="Add a bio..."></textarea>

Submit Button

The submit button is used to submit the form. 

<form>
    <input type="text">
    <input type="submit" value="submit the form">
</form>
