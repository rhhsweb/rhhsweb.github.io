Lists

HTML lists provide a way to display groups of items.

List Types

There are three list types in HTML: ordered lists (<ol>), unordered lists (<ul>), and definition lists (<dl>).

Ordered Lists

Ordered lists are lists used when the order of items is significant. List items are nested inside the <ol> element using the <li> element. 

<ol>
    <li>Whisk dry ingredients together.</li>
    <li>Mix wet ingredients with a stand mixer.</li>
    <li>Preheat oven to 300 fahrenheit.</li>
    <li>Add dry ingredients into stand mixer with wet ingredients.</li>
    <li>Pour mixture into mold.</li>
    <li>Bake in oven for 25 minutes.</li>
</ol>
  1. Whisk dry ingredients together.
  2. Mix wet ingredients with a stand mixer.
  3. Preheat oven to 300 fahrenheit.
  4. Add dry ingredients into stand mixer with wet ingredients.
  5. Pour mixture into mold.
  6. Bake in oven for 25 minutes.

There are a few attributes that control how the elements are labeled. The reversed boolean attribute labels the items in reverse order. 

<ol reversed>
    <li>Whisk dry ingredients together.</li>
    <li>Mix wet ingredients with a stand mixer.</li>
    <li>Preheat oven to 300 fahrenheit.</li>
    <li>Add dry ingredients into stand mixer with wet ingredients.</li>
    <li>Pour mixture into mold.</li>
    <li>Bake in oven for 25 minutes.</li>
</ol>
  1. Whisk dry ingredients together.
  2. Mix wet ingredients with a stand mixer.
  3. Preheat oven to 300 fahrenheit.
  4. Add dry ingredients into stand mixer with wet ingredients.
  5. Pour mixture into mold.
  6. Bake in oven for 25 minutes.

The type of label for the items can be set using the type attribute of the <ol> element.

Value Type
a lowercase letters
A uppercase letters
i lowercase roman numerals
I uppercase roman numerals
1 numbers (default) 
<ol type="a">
    <li>Whisk dry ingredients together.</li>
    <li>Mix wet ingredients with a stand mixer.</li>
    <li>Preheat oven to 300 fahrenheit.</li>
    <li>Add dry ingredients into stand mixer with wet ingredients.</li>
    <li>Pour mixture into mold.</li>
    <li>Bake in oven for 25 minutes.</li>
</ol>
  1. Whisk dry ingredients together.
  2. Mix wet ingredients with a stand mixer.
  3. Preheat oven to 300 fahrenheit.
  4. Add dry ingredients into stand mixer with wet ingredients.
  5. Pour mixture into mold.
  6. Bake in oven for 25 minutes.

The type attribute can also be added to individual <li> 

<ol type="a">
    <li type="A">Whisk dry ingredients together.</li>
    <li>Mix wet ingredients with a stand mixer.</li>
    <li type="A">Preheat oven to 300 fahrenheit.</li>
    <li>Add dry ingredients into stand mixer with wet ingredients.</li>
    <li type="A">Pour mixture into mold.</li>
    <li>Bake in oven for 25 minutes.</li>
</ol>

  3. Preheat oven to 300 fahrenheit.

Finally, the starting label can be changed with the start attribute on the <ol> element. 

<ol type="a" start="4">
    <li>Add dry ingredients into stand mixer with wet ingredients.</li>
    <li>Pour mixture into mold.</li>
    <li>Bake in oven for 25 minutes.</li>
</ol>
  1. Add dry ingredients into stand mixer with wet ingredients.
  2. Pour mixture into mold.
  3. Bake in oven for 25 minutes.

Unordered Lists

Unordered lists are lists used when the order of items is significant. 

<ul>
    <li>Penne</li>
    <li>Broccoli</li>
    <li>Olive oil</li>
    <li>Parmesan</li>
</ul>
  • Penne
  • Broccoli
  • Olive oil
  • Parmesan

Definition Lists

Definition lists are lists containing description terms (<dt>) and description details (<dd>) elements. 

<dl>
    <dt>HTML</dt>
    <dd>HTML is a language used to describe the structure of webpages.</dd>
    <dt>CSS</dt>
    <dd>CSS is a language used to describe the appearance of webpages.</dd>
    <dt>JavaScript</dt>
    <dd>JavaScript is a language used for adding functionality to webpages.</dd>
</dl>
HTML
HTML is a language used to describe the structure of webpages.
CSS
CSS is a language used to describe the appearance of webpages.
JavaScript
JavaScript is a language used for adding functionality to webpages.

Nesting Lists

Lists can be nested inside lists. 

<ul>
    <li>Sauce
        <ol>
            <li>Combine soy sauce with mirin.</li>
            <li>Add salt and pepper to taste.</li>
        </ol>
    </li>
    <li>Noodles
        <ol>
            <li>Set a large pot of water to a boil.</li>
            <li>Add noodles, cook for 8 minutes.</li>
            <li>Strain and rinse with cold water.</li>
        </ol>
    </li>
    <li>Garnish
        <ul>
            <li>Spring onions</li>
            <li>Sesame seeds</li>
        </ul>
    </li>
</ul>
  • Sauce
    1. Combine soy sauce with mirin and soup base.
    2. Add salt and sesame oil to taste.
  • Soba noodles
    1. Set a large pot of water to a boil.
    2. Add noodles, cook for 8 minutes.
    3. Strain and toss with cold water.
  • Garnish
    • Spring onions
    • Sesame seeds
