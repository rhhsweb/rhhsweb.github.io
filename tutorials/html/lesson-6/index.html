Comments

Comments are used to insert text that you don't want to be displayed on the webpage. 

<!-- this is a comment -->
<p>Hi there!</p>
<!-- this is
also 
a comment -->
<p>Hello <!-- this is another comment --> there!</p>

Hi there!

Hello there!

Anything between the <!-- and the --> tags will be ignored by the browser. This is useful for leaving notes as a programmer about your code.
