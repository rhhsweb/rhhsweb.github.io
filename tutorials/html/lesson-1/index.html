Document Structure

All HTML documents start with the same basic layout. This consists of a doctype and <html>, <head>, and <body> elements. Let's take a look it: 

<!DOCTYPE html>
<html>
    <head>
        <!-- Comment: This is where website metadata goes -->
    </head>
    <body>
        <!-- Comment: This is where visible website content goes -->
    </body>
</html>

Doctype

The doctype is a bit of information that goes at the beginning of all HTML documents. It tells the browser about the type and version of the file it is expected to run. It is not actually an HTML element.

<!DOCTYPE html> tells the browser to run an HTML 5 document, which is what we want.

It is also possible to run older versions of HTML using a different doctype. For example, if we wanted to run an HTML file using HTML 4.01, it would look like this: 

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

There is little reason to use an older version of HTML now. These doctype declarations allow older HTML files and websites to continue working.

HTML Element

The <html> element wraps all the content of the website. It is also known as the root element.

Head Element

The <head> element contains important data about the website that are not part of the visible structure of the page.

For example, the <title> element contains the title of the website displayed on the tab. 

<head>
    <title>My Website<title>
</head>

Or, the <meta> element which can contain a variety of information, including the page author (you!). 

<head>
    <meta name="author" content="James Smith">
</head>

Body Element

The <body> element contains all of the elements relating to the structure of the page. There can only be one body per page. 

<body>
    <h1>My <i>Three</i> Favourite Colours</h1>
    <ul>
        <li>Red</li>
        <li>Blue</li>
        <li>Yellow</li>
    </ul>
</body>

Complete Example

<!DOCTYPE html>
<html>
    <head>
        <title>My Website<title>
        <meta name="author" content="James Smith">
    </head>
    <body>
        <h1>My <i>Three</i> Favourite Colours</h1>
        <ul>
            <li>Red</li>
            <li>Blue</li>
            <li>Yellow</li>
        </ul>
    </body>
</html>
