Tables

The <table> element alows you to display a table in HTML. 

<table>
    <tr>
        <th>First Name</th>
        <th>Last Name</th>
        <th>Email</th>
    </tr>
    <tr>
        <td>John</td>
        <td>Smith</td>
        <td>johnsmith@email.com</td>
    </tr>
    <tr>
        <td>Jamie</td>
        <td>Sun</td>
        <td>jamie_sun@email.com</td>
    </tr>
    <tr>
        <td>Liz</td>
        <td>Lee</td>
        <td>liz.lee23@email.com</td>
    </tr>
</table>
First Name Last Name Email
John Smith johnsmith@email.com
Jamie Sun jamie_sun@email.com
Liz Lee liz.lee23@email.com

Table data for a cell is stored in a <td> element, which is nested within a table row (<tr>). Table headers (<th>) are also nested within their own table row element.

Merging Cells

In order to make a cell take up more than one cell width of space, the colspan attribute can be used. 

<table>
    <tr>
        <th colspan="3">Header</th>
    </tr>
    <tr>
        <td>one</td>
        <td>two</td>
        <td>three</td>
    </tr>
    <tr>
        <td>one</td>
        <td colspan="2">two</td>
    </tr>
    <tr>
        <td colspan="3">one</td>
    </tr>
</table>
